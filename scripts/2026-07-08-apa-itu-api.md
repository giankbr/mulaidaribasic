## Body (3–46 detik)

### Poin 1
**Judul:** API itu ibarat pelayan di restoran
**Isi:** Bayangkan lo pesen makanan. Lo nggak masuk ke dapur langsung kan? Lo pesen ke pelayan, terus pelayan nyampaikan ke koki. Nah, API itu perannya sama persis kayak pelayan tadi.

### Poin 2
**Judul:** Jembatan penghubung antar dua aplikasi
**Isi:** Tugas utamanya adalah nganterin pesanan dari aplikasi ke server, lalu ngebawain balasan dari server ke aplikasi lo lagi. Semua proses ini jalan terus secara otomatis tanpa lo sadari.

### Poin 3
**Judul:** Bikin fitur jadi jauh lebih cepat dibikin
**Isi:** Dengan API, developer nggak perlu bikin semuanya dari nol. Contohnya mau pasang login Google atau pembayaran, tinggal pakai API yang udah disediain. Praktis banget kan buat ngebangun aplikasi.
