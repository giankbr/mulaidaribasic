## Body (3–46 detik)

### Poin 1
**Judul:** API ibarat jembatan antara kamu dan server
**Isi:** Pas kamu pesan makanan, kamu nggak masuk ke dapur. Kamu panggil pelayan, terus pesananmu dikirim ke koki. Pelayan ini peran API yang menyambungkan aplikasi dengan database.

### Poin 2
**Judul:** REST cuma aturan standar buat komunikasi data
**Isi:** Bayangkan REST itu tata krama saat ngobrol. Semua orang ikuti aturan yang sama, jadi aplikasi dari bahasa pemrograman berbeda bisa saling paham. Hasilnya, data bolak-balik jadi jauh lebih gampang.

### Poin 3
**Judul:** Pakai HTTP method buat ngasih perintah server
**Isi:** Ada GET buat ambil data, POST buat bikin data baru, PUT buat update, dan DELETE buat hapus. Semua perintah standar ini dikirim lewat internet dengan format yang rapi.
