## Hook (0–4 detik)
**Headline:** Ketik nama website di browser, tapi nggak ngerti prosesnya?  
**Subtitle:** Di balik layar, ada sistem yang mencari alamat server asli dari nama website itu. Sistem ini namanya DNS, dan kerjanya simpel banget.

## Body (3–46 detik)

### Poin 1
**Judul:** DNS itu ibarat buku telepon internet
**Isi:** Komputer pakai angka buat berkomunikasi lewat alamat IP. Tapi kamu pasti lebih ingat nama website seperti google.com. DNS bikin kamu nggak perlu hafal angka panjang itu.

### Poin 2
**Judul:** Browser bertanya langsung ke server DNS
**Isi:** Saat kamu buka situs, browser langsung ngirim pertanyaan ke server DNS. Tugasnya cuma satu, nyocokin nama domain kamu dengan alamat IP server yang benar.

### Poin 3
**Judul:** IP ketemu, koneksi ke server dibuka
**Isi:** Setelah server DNS kasih alamat IP yang tepat, browser kamu pakai angka itu buat ngobrol langsung. Server web balas, dan halaman website pun langsung kebuka.

Sering banget kita ngetik nama website di browser tanpa ngerti gimana prosesnya sampe bisa kebuka. Sebenarnya, ada sistem penting yang namanya DNS atau Domain Name System.
