## Slide 2
**Headline:** API kayak|**pelayan** restoran
**Subtitle:** Lo nggak masuk dapur langsung kan? Lo pesen ke pelayan, pelayan sampaikan ke koki. API punya peran mirip: nganterin permintaan dan bawa balasan.

**Detail:**

## Slide 3
**Headline:** Request|**dan** response
**Subtitle:** Aplikasi lo kirim request ke server lewat API. Server balas dengan response. Dua arah ini jalan otomatis setiap kali fitur dipakai.

- 200 OK
**Detail:**
- Request = permintaan data
- Response = jawaban dari server
