# Copy Rules — @mulaidaribasic

Wajib dibaca sebelum generate script / microblog.

## Prinsip
- Tulis seperti educator personal, bukan brosur agency
- Satu slide/scene = satu konsep basic
- Kalimat pendek, jelas, humble
- Bahasa Indonesia natural; istilah tech boleh (HTTP, Git, API) tanpa diapit asterisk
- Boleh pakai gaya **Jaksel lo/gue** yang santai, tapi tetap jelas dan nggak alay
- Jangan pakai: "dampaknya", "solusinya adalah", "di Novagen kami", "digital engineering partner"
- **Punctuation:** kalimat normal pakai koma/titik. **Jangan** em dash (`—`), double dash (`--`, `---`)

## Contoh BAGUS vs JELEK

**Hook jelek:** HTTP request adalah mekanisme fundamental dalam arsitektur web modern yang wajib dipahami developer.

**Hook bagus:** Kirim request ke server, tapi nggak ngerti apa yang terjadi? Mulai dari sini.

---

**Poin jelek:** Request dan response merupakan komponen esensial dalam komunikasi client-server yang harus dikuasai.

**Judul bagus:** Request = pesan dari browser  
**Isi bagus:** Browser kirim HTTP request ke server. Server baca, proses, lalu balikin response. Dua arah, satu siklus.
